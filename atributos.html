Atributos

  • id: Define um identificador único para o elemento.
  • name: Nome do campo, usado para identificar os dados enviados.
  • class: Define uma ou mais classes CSS para estilização.
  • style: Adiciona regras de estilo inline.
  • disabled: Desabilita o campo, tornando-o inacessível ao usuário.
  • required: Torna obrigatório selecionar um arquivo antes de enviar o formulário.
  • multiple: Permite o upload de vários arquivos.
  • accept: Especifica os tipos de arquivos permitidos (por exemplo, imagens, vídeos, documentos).
      • Valores comuns utilizados no atributo accept:
    • "image/*": Apenas imagens.
    • "image/*": Apenas imagens.
    • "audio/*": Apenas arquivos de áudio.
    • .pdf, .docx, .txt: Tipos de arquivos específicos.

Exemplos de Atributos do input type="file"

1. Atributo id

<input type="file" id="basic">

2. Atributo name

<input type="file" id="nameExample" name="imagem">

3. Atributo accept

<input type="file" id="acceptExample" accept="image/*">

4. Atributo multiple

<input type="file" id="multipleExample" multiple>/div>

5. Atributo required

<input type="file" id="requiredExample" required>

6. Atributo disabled

<input type="file" id="disabledExample" disabled>

7. Combinação de Atributos

<input type="file" id="combinedExample" name="documento" accept=".pdf" required>
